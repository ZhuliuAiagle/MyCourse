# infocrawler
- 这是一个INTELLIJ IDEA工程目录，由最新版IDEA生成，建立使用IDEA打开；
- 本工程使用maven配置包管理；
- 本工程的.class文件在target文件夹的对应包下；
- 爬虫的源代码为App.java, 搜索引擎的源代码为LuceneTest.java。它们的.class文件同名；
- 爬虫使用的代理可能已经失效（租用到期），届时相当于只有本机一台服务器在执行爬虫；
- 工程根目录中的javaio和index分别为数据集和索引文件，默认每次运行搜索之前的索引都会被自动删去；
- 在工程目录之外，我提供了单独的数据集，可供参考。但实际上程序运行时使用的是工程内部的那个数据集。

- Github版本说明：数据集因为太大了已经被我删掉了，所以LuceneTest无法直接运行。代码仅供参考
